How to add as an app in your phone:

IOS: 
1. Open safari
2. Go to rollbasebjj.github.io
3. Click the "..." on the bottom right
4. Click "Share"
5. Scroll down until you see "Add to Home Screen"
6. Make sure "Open as Web App" is GREEN
7. Click "Add"
8. Done!

Downside: 
Safari downloads the website and runs the code locally. And because of such, installing any future update will require you to manually copy your data from
the old app to the new app.
